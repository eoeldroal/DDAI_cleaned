# DDAI - Document-based Dialog AI

GSPO (Generative Search Policy Optimization) training framework for Vision-Language Models.

## Features

- Multi-turn RAG (Retrieval-Augmented Generation) for document understanding
- VLM as Judge reward model with Gemini Flash
- Based on verl (Volcano Engine Reinforcement Learning)

## Installation

```bash
pip install -e .
```

## Usage

```bash
# GSPO Phase 2 Training with Gemini Flash
bash gspo_phase2_gemini_flash.sh
```
